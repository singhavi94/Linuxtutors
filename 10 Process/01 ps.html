Process

  • ps



Ps :


        Every work going on computer is done process. Process is small executing task which handles working of a system.`ps` command is used to see the processes going on at that time. It creates a snapshot of all processes at that time.

Synopsis :


ps [options]

Options :

[ -a ] :It selects all process

[ -e ] :It is similar to a options.

[ -r ] :It restrict to running process only.

[ -C {cmdlist} ] :It list all process started by cmdlist commands.

[ -p ] :It selects process by pid(process id).

You can find more on man pages in detail.
